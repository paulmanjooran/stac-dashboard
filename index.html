⚙️ STEP 2 — Deploy Google Apps Script

One-time setup
  1. Open your Google Sheet → click Extensions in the top menu → Apps Script
  2. In the Apps Script editor: delete all existing code
  3. Click 📋 Copy Script button below → paste into the editor (Ctrl+V)
  4. Click the 💾 Save icon (or Ctrl+S)
  5. Click Deploy → New Deployment
  6. Click the ⚙ gear icon next to "Select type" → choose Web App
  7. Description: STAC v5 | Execute as: Me | Who has access: Anyone
  8. Click Deploy
  9. Click "Authorize access" → Review permissions → Advanced → "Go to STAC (unsafe)" → Allow
  10. A URL appears ending in /exec — copy it
  11. Paste the URL in the field below → click Save
  12. Click 🔌 Test Connection — must show ✅ Connected
  13. Click ⚡ Auto-Add Headers — adds columns S–AF to your sheet automatically
⚠️ Important: Every time you edit the Apps Script code you must create a New Deployment (not update existing). The URL changes each time.
